Prefacio

Esta guía está diseñada para llevarte desde los fundamentos más básicos de Python hasta conceptos avanzados de programación. Está dirigida a principiantes sin conocimientos previos de programación y a programadores que buscan aprender Python.

Capítulo 1: Fundamentos de Python

1.1 ¿Qué es Python?

Historia del Lenguaje

  • Creación: Desarrollado por Guido van Rossum en 1991
  • Nombre: Inspirado en el grupo de comedia británico Monty Python
  • Evolución: Desde su creación, ha pasado por varias versiones, siendo Python 3.x la actual y recomendada

Filosofía y Características Principales

  • Zen de Python: "Simple es mejor que complejo"
  • Enfocado en la legibilidad y productividad del código
  • Comunidad activa y en constante crecimiento

Casos de Uso

  • Desarrollo Web: Django, Flask
  • Ciencia de Datos: NumPy, Pandas, SciPy
  • Inteligencia Artificial: TensorFlow, PyTorch
  • Automatización: Scripting, testing
  • Empresas que lo utilizan: Google, Netflix, Spotify, NASA

1.2 Características Únicas de Python

Aspectos Técnicos

  • Lenguaje Interpretado: No requiere compilación previa
  • Tipado Dinámico y Fuerte: Variables pueden cambiar de tipo, pero con chequeo de tipos
  • Multiplataforma: Funciona en Windows, macOS, Linux
  • Sintaxis Clara: Legible y cercana al lenguaje humano

Paradigmas de Programación

  • Programación Orientada a Objetos
  • Programación Funcional
  • Programación Imperativa
  • Programación Estructurada

Capítulo 2: Preparación del Entorno de Desarrollo

2.1 Instalación de Python

Pasos de Instalación

  1. Visitar el sitio oficial de Python (python.org)
  2. Descargar la última versión de Python 3.x
  3. Seguir el asistente de instalación
    • Windows: Marcar "Add Python to PATH"
    • macOS: Usar instaladores para sistema operativo
    • Linux: Usar gestores de paquetes (apt, yum)

Gestión de Versiones

  • pyenv: Administrar múltiples versiones de Python
  • virtualenv: Crear entornos virtuales aislados
  • Conda: Gestión de entornos y paquetes

Capítulo 7: Programación Orientada a Objetos

7.1 Conceptos Básicos


# Clases y Objetos
class Persona:
    def __init__(self, nombre, edad):
        self.nombre = nombre
        self.edad = edad
    
    def presentarse(self):
        return f"Soy {self.nombre} y tengo {self.edad} años"

# Herencia
class Estudiante(Persona):
    def __init__(self, nombre, edad, carrera):
        super().__init__(nombre, edad)
        self.carrera = carrera
    
    def estudiar(self):
        return f"{self.nombre} está estudiando {self.carrera}"

7.2 Encapsulamiento


class Cuenta:
    def __init__(self, saldo_inicial):
        self._saldo = saldo_inicial
    
    @property
    def saldo(self):
        return self._saldo
    
    def depositar(self, monto):
        if monto > 0:
            self._saldo += monto
    
    def retirar(self, monto):
        if 0 < monto <= self._saldo:
            self._saldo -= monto

def dividir(a, b):
    try:
        resultado = a / b
    except ZeroDivisionError:
        print("Error: División por cero")
        return None
    except TypeError:
        print("Error: Tipo de dato inválido")
        return None
    else:
        print("División exitosa")
        return resultado
    finally:
        print("Operación de división completada")

10.1 Proyecto: Gestor de Tareas


class GestorTareas:
    def __init__(self):
        self.tareas = []
    
    def agregar_tarea(self, descripcion):
        tarea = {
            'descripcion': descripcion,
            'completada': False
        }
        self.tareas.append(tarea)
    
    def listar_tareas(self):
        for i, tarea in enumerate(self.tareas, 1):
            estado = "✓" if tarea['completada'] else " "
            print(f"{i}. [{estado}] {tarea['descripcion']}")
    
    def marcar_completada(self, indice):
        if 0 < indice <= len(self.tareas):
            self.tareas[indice-1]['completada'] = True

10.2 Proyecto: Juego de Adivinar Número


import random

def juego_adivinar_numero():
    numero_secreto = random.randint(1, 100)
    intentos = 0
    
    print("¡Bienvenido al juego de adivinar el número!")
    print("Tengo un número entre 1 y 100. ¡Adivina!")
    
    while True:
        try:
            intento = int(input("Introduce tu número: "))
            intentos += 1
            
            if intento < numero_secreto:
                print("Demasiado bajo. ¡Intenta de nuevo!")
            elif intento > numero_secreto:
                print("Demasiado alto. ¡Intenta de nuevo!")
            else:
                print(f"¡Felicidades! Adivinaste el número en {intentos} intentos.")
                break
        
        except ValueError:
            print("Por favor, introduce un número válido.")
